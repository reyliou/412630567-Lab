# 尼歐計畫 (Project Neo) - 初始化連線介面

系統偵測到未經授權的終端連線... 正在建立安全通道...

## 🌐 存取路徑

*   **入口節點**：`http://<伺服器_IP>:8080`
*   **目標網路**：NEO-MALL 內部骨幹

## 🌐 開放服務

*   **Port 8080**：Nginx Gateway (主要訪問入口，配置了 WAF 與反向代理)
*   **Port 8081**：NEO-MALL Internal 管理埠 (繞過網關直連後台，路徑穿越關鍵點)
*   **Port 21**：企業內部備份 FTP 伺服器 (匿名存取已啟用)


## 🎯 最終協定

您的任務只有一個：**取得系統最高權限 (Root Privilege)**。
所有隱藏的資訊、憑據與路徑均散落在系統的各個層級中。

## 🚩 識別碼 (Flag)

*   **唯一標識格式**：`FLAG{學號_隨機雜湊}`
*   **位置情報**：
    1.  **隱藏配置**：系統核心配置區中隱藏著初步的存取憑證。
    2.  **一般使用者層級**：透過特定系統操作後，於 `neo-user` 使用者家目錄生成。
    3.  **核心禁區 (Root)**：系統最深處的管理權限區。
*   **注意**：部分識別碼需透過特定的系統操作後方可觸發生成，你可以用curl打天下，也可以使用我提供的診斷工具，雖然很難用就是了。
